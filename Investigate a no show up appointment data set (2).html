#set y label
ax.set_ylabel("No. of no show up", fontsize=14);
#set y value
ax.set_yticks([0, 2000, 4000, 6000, 8000, 10000, 12000, 14000])
#set x label
ax.set_xlabel('Gender', fontsize=14)
# create a list to collect the plot data
totals = []
# find the values and append to list
for i in ax.patches:
    totals.append(i.get_height())
# set individual bar lables using above list
total = sum(totals)
for i in ax.patches:
    # get_x pulls left or right; get_height pushes up or down
    ax.text(i.get_x()+.12, i.get_height()-3, \
            str(round((i.get_height()/total)*100))+'%', fontsize=12,
                color='black')

Conclusions¶

No show up data set contain 110527 row and 14 column. The data set didn't have missing or duplicate value but contain invalied value like in age number(contain -1, and 0 value) and in appointment day the appointment day come before schaduled day. For that the data fram size which used in analysis contain 69828 rows and 14 column.

In the beginning of the research we asked in which age the patient no show up? and in which gender the patients no show up? Number of attended appointment is 49894 which the precentage represent 71%, and the number of did't attend appointment is 19934 which is represent 29%. The age mostly no show up from 25 to 35, on other hand the age from 50 to 60 mostly show up. Thw gender mostly no show up was in female by 13380 which represent 67%, on other hand the number of Male is 6554 which represent 33%.

In conclusion, the patients no show up represent 29% of the clean dataset. The age from 25 to 35 mostly no show up, and the most of no show up was in female.
